# Content Review Checklist
**Purpose:** Verify all content before publication  
**Used by:** All team members before shipping

---

## Pre-Publish Checklist

### Message & Clarity
- [ ] Message is immediately clear (no ambiguity)
- [ ] Can stand alone without verbal explanation
- [ ] Clearest version of the idea (no way to simplify further)
- [ ] No unnecessary jargon or inside language
- [ ] Tone matches brand (firm = measured; Jason = gritty/peer-to-peer)

### Authority Check
- [ ] Doesn't sound promotional or self-congratulatory
- [ ] Avoids over-explaining credentials
- [ ] Sounds confident, not defensive
- [ ] Would stand up in a trial context (or makes sense for personal brand)
- [ ] Authority feels quiet (not trying to impress)

### Length & Structure
- [ ] Shorter version would work? (Make it shorter)
- [ ] Says the thing once, not repeated in different words
- [ ] No padding for SEO or page length
- [ ] Sections don't exist "just in case"
- [ ] One clear CTA (if needed)

### For Website Content Specifically
- [ ] Location references reflect reality (no aspirational geography)
- [ ] Metadata accurately represents the page
- [ ] No keyword stacking
- [ ] Geographic frame is clear (one primary location)
- [ ] Content + location together = user gets what they need

### For Social Media Specifically
- [ ] Authentic to Jason (not manufactured)
- [ ] Fits one of the four content pillars (Rider's Life, Gourmet, Cigars/Sports, Alligator Wisdom)
- [ ] Caption is clear and compelling
- [ ] CTA is natural, not forced
- [ ] Visual supports the message
- [ ] Hashtags are relevant (3–5 max)

### Visual Check (If Applicable)
- [ ] Reinforces message
- [ ] Matches aesthetic system
- [ ] Doesn't distract from content
- [ ] Intentional, not decorative
- [ ] Could the message work without it? (If yes, visual earns its place)

### Doctrine Alignment
- [ ] Serves the North Star (trust, clarity, outcomes)
- [ ] Doesn't violate Operating Manual
- [ ] Follows Execution Standards
- [ ] Matches Brand Authority rules
- [ ] Respects SEO/Location rules (firm website)

---

## Red Light / Green Light

### GREEN LIGHT → Publish
✅ Clear, intentional, purposeful  
✅ Passes all above checks  
✅ Authority feels quiet  
✅ Can defend the decision without over-explaining  

### RED LIGHT → Revise
❌ Feels promotional or self-serving  
❌ Requires explanation after delivery  
❌ Changes are cosmetic, not meaningful  
❌ Violates any doctrine rule  
❌ Seems driven by anxiety, not outcomes  

### YELLOW LIGHT → Ask
⚠️ Not sure if it's clear enough?  
⚠️ Unsure about tone?  
⚠️ Unsure if it's finished?  

→ Get a second opinion from someone who knows the brand

---

## Quick Reference: Decision Framework

**When in doubt:**
1. Does this improve understanding, authority, or outcomes?
2. Is this the clearest version?
3. Would I defend this calmly, without over-explaining?

If **yes** to all three → Publish  
If **no** to any → Revise or don't publish
