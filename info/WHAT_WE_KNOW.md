## FIRM WEBSITE PERFORMANCE (From Search Console Data)

### Key Metrics
- **Total top-page impressions:** 500k+ monthly
- **Total top-page clicks:** ~3,700 monthly
- **Average CTR:** 0.59%

### What's Working
- Blog content: 1.5–3.7% CTR (high-intent content)
- Q&A format: Strong performance
- Homepage: 0.59% CTR

### What's Not Working
- Location pages: 0.01–0.35% CTR (critical issue)
- Location + practice area: Often 0 clicks
- Attorney pages: 0.88% CTR (underperforming)

### Issues Identified
- 650k+ impressions from location pages = ~900 clicks
- Blog content: 75k impressions = ~1,300 clicks
- **Conversion Ratio:** Blog converts 12x better than location pages
- **Root Cause:** Site structure prioritizes SEO volume over user intent
