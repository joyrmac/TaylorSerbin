## Escalation Path

**If a question comes up:**

1. **Does it violate doctrine?** → Refer to `/docs/`
2. **Is it a tone/voice issue?** → Ask Reviewer
3. **Is it a legal/accuracy issue?** → Ask Firm Contact
4. **Is it a strategic question?** → Refer to `/strategy/`
5. **Is it a technical/platform issue?** → Research or escalate to platform support

## Communication Norms

### Jason → Social Manager
- **Frequency:** Weekly content batch (or as it happens)
- **Format:** Photos + simple context ("riding in mountains," "escargot prep")
- **Method:** Email, WhatsApp, or agreed method
- **Turnaround:** Social Manager responds within 24 hours

### Social Manager → Reviewer
- **Frequency:** Before each post (or batch of posts)
- **Format:** Draft caption + visual + metadata
- **Method:** Shared doc or email
- **Turnaround:** Reviewer responds within 24 hours

### Social Manager → Analytics/Reporting
- **Frequency:** Monthly
- **Format:** Report on engagement, trends, recommendations
- **Method:** Shared spreadsheet or summary doc

### All → Team
- **Weekly Sync (Optional):** 15-min check-in on content, metrics, blockers
- **Monthly Review:** Full strategy review + performance analysis
- **Documentation:** All decisions logged in git commits
